proto
================================================================================

SNS via Ruby on Rails

はじめに
--------------------------------------------------------------------------------

protoはrailsで作られたSNSエンジンです．
単体ではなくカスタマイズして使うことを前提にしています．


protoの機能
--------------------------------------------------------------------------------

- ユーザ認証
- ユーザプロフィール
- ポスト(ブログ)
- メッセージ
- ユーザをフォロー
